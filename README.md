# Spam-message-checker
Developed a spam message classification system using Python, leveraging a self-created dataset. The system accurately distinguishes between spam and legitimate messages and supports real-time prediction through an interactive interface built using Streamlit.

## How to Run
1. Clone the repository:
git clone https://github.com/rajanv29/Spam-message-checker.git

2. Navigate to the folder:
cd Spam-message-checker

3. Install dependencies:
pip install -r requirements.txt

4. Run the app:
streamlit run app.py

Note: Other meathod to open

        -Install the requirements in the terminal.
        -run the app using the command: "streamlit run app.py"
